# מבחני הכנה ליחידות טכנולוגיות

פלטפורמת אימון אינטראקטיבית להכנה לגיוס ליחידות טכנולוגיות נבחרות בצבא.

**[כניסה לאתר](https://vyaron.github.io/coding-academy-pre-tech-units/)**

---

## תכונות

- **סוגי שאלות מגוונים** – בחירה יחידה, אמת/שקר, קוד, סידור פריטים ושאלון אישיות
- **ניקוד חי** – ציון מתעדכן בזמן אמת
- **טיימר** – ספירה לאחור עם אינדיקטור חזותי (ירוק → צהוב → אדום)
- **סיפור הקשר** – מבחן כלל חמן כולל סיפור בנייה הנפתח בכפתור ייעודי
- **שמירת התקדמות** – מצב המבחן נשמר ב-sessionStorage וניתן להמשיך ממקום שנעצרתם
- **דגל שאלה** – סמן שאלות לחזרה לפני הגשה
- **עמוד תוצאות** – סיכום מפורט עם פירוט כל תשובה
- **בלוג** – מאמרים בנושאי הכנה למיונים, סייבר, תכנות ויחידות טכנולוגיות
- **עיצוב רספונסיבי** – עובד במובייל ובדסקטופ

---

## טכנולוגיות

- **React 19** + **TypeScript**
- **Vite** – בנייה וסביבת פיתוח
- **React Router v7** (HashRouter לתאימות GitHub Pages)
- **@dnd-kit** – גרירת שאלות לסידור (שאלות מסוג `order`)
- **CSS** – ללא ספריות UI, עיצוב מותאם אישית עם CSS custom properties
- **GitHub Actions** – CI/CD אוטומטי לעמוד GitHub Pages

## הוספת מבחן חדש

1. צור קובץ JSON ב-`public/exams/`:

```json
{
  "id": "my-exam",
  "title": "שם המבחן",
  "description": "תיאור קצר",
  "duration": 1200,
  "passingScore": 75,
  "questions": [
    {
      "id": "q1",
      "type": "truefalse",
      "text": "טקסט השאלה",
      "answer": true,
      "explanation": "הסבר לתשובה"
    },
    {
      "id": "q2",
      "type": "single",
      "text": "טקסט השאלה",
      "options": ["א", "ב", "ג", "ד"],
      "answer": 0,
      "explanation": "הסבר לתשובה"
    }
  ]
}
```

סוגי שאלות: `single` | `truefalse` | `code` | `order` | `personality`

2. הוסף את המבחן למערך `EXAMPLE_EXAMS` ב-`HomeScreen.tsx`
3. הוסף את ה-key למערך `QUIZ_CONFIGS` ב-`QuizIntroScreen.tsx`
